API mapping (available today or via Experience layer)

  • Overview tier & rewards — GET /v1/rewards/summary/referrers/{ucn}
  • Primary nudge — GET /v1/recommendations/{trackId} → rendered cards
  • Leaderboard snippet — GET /v1/tenants/{tenant}/leaderboards/{code}/me
  • Refer & Earn — POST /referrals/bootstrap → POST /referrals/codes
  • Share link / QR — channel BFF builds /friend?referral_code=…
  • Progress list — GET /v1/referrers/{ucn} (normalized to milestones)
  • Missions — GET /v1/missions/referrer/{ucn}
  • Terms — POST /referrals/accept-terms
  • Referee validate — POST /public/referrals/validate
  • Referee accept — POST /v1/progress (ACCOUNT_OPENED)
  • Referee next steps — GET /v1/recommendations/{trackId}
  • Identity — x-ocep-ucn, x-ocep-tenant (gateway → BFF, never shown to user)
