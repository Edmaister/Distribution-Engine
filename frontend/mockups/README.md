# Customer experience mocks (HTML)

Interactive **web + mobile** prototypes for the target referrer and referee journeys.

## Open locally

```cmd
cd /d "C:\Projects\Referral Engine\frontend\mockups"
start index.html
```

Or double-click `index.html` in File Explorer.

## What's inside

| Control | Screens |
|---------|---------|
| **Referrer** | Overview · Refer & Earn · Progress · Tips & Tools · Terms (first visit) |
| **Referee** | Enter code · Validated · Success + recommendations |

- **Left:** mobile app (phone frame — WebView / native shell)
- **Right:** web banking portal (sidebar nav + wider layout)

Each screen shows **API tags** mapping to your existing Referral Engine endpoints and the recommended Experience layer.

## Files

- `index.html` — interactive hub (no build step)
- `cx-mock.css` — design tokens aligned with channel BFF / FNB mobile shell

These are **static mocks** for CX/UX review and stakeholder demos, not wired to a live API.
