  1. Golden-path and negative-path contract tests are in place.

Phase 3: Attribution Operations

Make attribution explainable and supportable end to end.

  1. Unified attribution trace from code/link to outcome.
  2. Operator investigation and support-case workflow.
  3. Tenant-safe reporting and exports.
  4. Audit/idempotency hardening for every exposed command.

Phase 4: Production + DLaaS

Prove live confidence, then expand distribution depth.

  1. Complete live DB/state verification and migration drift checks.
  2. Golden-path and failure-path SaaS E2E tests.
  3. Then attach commissions, funding, fulfilment, settlement, and billing.
  4. Keep marketplace, white-label, and broad partner depth as DLaaS expansion.

SaaS Guardrails

These prevent the focused SaaS from becoming a brittle one-off app.

Do Not Invent State

Use database schema and service-layer code as source of truth. New statuses need migration, service, API, UI, and tests.

Separate Money Depth

Rewards can be visible as status, but deep funding, fulfilment, settlement, and wallet operations should stay bounded at first.

Make Every Command Idempotent

Campaign creation, link issuance, event ingestion, attribution, replay, and webhook delivery need duplicate protection.

Audit the Spine

Sensitive setup, state transitions, overrides, replays, and attribution decisions need actor, reason, before, after, and trace.

Separate DLaaS Compatibility Notes

These are not launch requirements for the first SaaS wedge. They are naming and boundary choices that keep the code reusable later.

Use Portable Names

Prefer account, campaign, participant, link/code, event, attribution trace, and status report over page-specific concepts.

Keep Money Optional

Referral reward summaries can stay visible, but funding, fulfilment, settlement, wallets, and commissions should not block the referral and attribution SaaS launch.

Expose Stable Contracts

Public APIs, webhooks, audit evidence, and event contracts should be narrow enough for the SaaS wedge and stable enough to expand.
